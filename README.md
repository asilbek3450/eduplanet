### Diplom ishi uchun yaratilgan EduPlanet dasturiy ilovasi.


## Installation
### 1. Clone this repository
```bash
git clone
```
### 2. Install requirements
```bash
pip install -r requirements.txt
```
### 3. Run the project
```bash
python manage.py runserver
```
### 4. Open the browser and go to http://
```
http://
```
